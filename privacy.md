Privacy Policy

Effective date: 26-03-2026

Who we are
I (Soufiane Marghabi) provide the “ADO-README” app for Atlassian Confluence.

What the app does
The app allows a Confluence site to display README or any md content from an Azure DevOps repository chosen by the customer.

Data we collect and process
The app processes:
	•	Azure DevOps organization name
	•	Azure DevOps project name
	•	Azure DevOps repository name
	•	branch name
	•	README or md file path
	•	Azure DevOps Personal Access Token (PAT)
	•	rendered README content requested by the customer

How we use this data
We use this data only to:
	•	store the app configuration selected by the customer
	•	authenticate to Azure DevOps using the customer-provided PAT
	•	fetch and display README content inside Confluence

Storage
The Azure DevOps PAT is stored using Atlassian Forge hosted storage and is intended to be stored securely as an app secret. The app configuration values are stored as part of app configuration and app storage.

Data sharing
We do not sell customer data.
We do not share customer data with third parties except as necessary to operate the app through Atlassian Forge and Azure DevOps.

Data retention
Configuration and secrets remain stored until the customer updates or removes them, or uninstalls the app.

Security
We use Atlassian Forge hosted infrastructure and storage features provided by Atlassian. Customers are responsible for managing and rotating their Azure DevOps PATs.

Customer controls
Customers can update or remove configuration and rotate the PAT at any time through the app configuration.
